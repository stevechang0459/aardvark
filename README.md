**Compile the source code**
---

```bash
# Create dependency files for the source code
make depall
# Compile the source code
make
```

---

```bash
# AStyle
$(pwd)/Astyle/bin/astyle.exe --options=$(pwd)/_astylerc -R ./*.c,*.h --exclude=AStyle --formatted
```
